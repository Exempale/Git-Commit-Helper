Git Commit HelperИнтерактивная консольная утилита, которая помогает писать красивые, понятные и стандартизированные сообщения для коммитов в стиле Conventional Commits.Больше не нужно вспоминать, какой префикс использовать (feat, fix или chore) и как правильно оформить многострочное описание. Скрипт сам задаст нужные вопросы и сформирует идеальный коммит.ОсобенностиИнтерактивное меню: Удобный выбор типа коммита по номеру.Защита от ошибок: Проверяет, находитесь ли вы в git-репозитории и есть ли файлы в stage (индексе).Conventional Commits: Строгое следование стандарту (тип, scope, описание, тело, footer).Превью перед сохранением: Показывает итоговое сообщение перед выполнением команды git commit.Без зависимостей: Написано на чистом Python, использует только стандартные библиотеки. Не требует установки тяжелых пакетов.УстановкаУтилита состоит всего из одного файла и легко интегрируется в вашу систему.Скачайте скрипт в удобную для вас директорию (например, ~/scripts/):mkdir -p ~/scripts
curl -o ~/scripts/git_commit_helper.py https://raw.githubusercontent.com/ВАШ_НИК/ВАШ_РЕПОЗИТОРИЙ/main/git_commit_helper.py
Сделайте файл исполняемым (для Linux / macOS):chmod +x ~/scripts/git_commit_helper.py
Создайте алиас (alias) для быстрого вызова. Добавьте следующую строку в ваш ~/.bashrc или ~/.zshrc:alias gch="python3 ~/scripts/git_commit_helper.py"
Обновите конфигурацию терминала:source ~/.bashrc  # или source ~/.zshrc
ИспользованиеВнесите изменения в вашем проекте.Добавьте файлы в индекс с помощью git:git add .
Запустите утилиту:gch

1. feat       - Новая функциональность

--- Детали коммита ---
Область / Scope (опционально, Enter для пропуска): ui
Краткое описание (обязательно): добавлена кнопка сохранения
Подробное описание / Body (опционально, Enter для пропуска):
> Кнопка теперь меняет цвет при наведении и поддерживает темную тему.
Связанные задачи / Footer (напр. Closes #123, опционально):
> Closes #42

--- Превью коммита ---
feat(ui): добавлена кнопка сохранения

Кнопка теперь меняет цвет при наведении и поддерживает темную тему.
