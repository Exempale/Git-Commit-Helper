# Git Commit Helper

Интерактивная консольная утилита, которая помогает писать красивые, понятные и стандартизированные сообщения для коммитов в стиле **Conventional Commits**.

Больше не нужно вспоминать, какой префикс использовать (`feat`, `fix` или `chore`) и как правильно оформить многострочное описание. Скрипт сам задаст нужные вопросы и сформирует идеальный коммит.

---

## Особенности

* **Интерактивное меню:** Удобный выбор типа коммита по номеру.
* **Защита от ошибок:** Проверяет, находитесь ли вы в git-репозитории и есть ли файлы в stage (индексе).
* **Conventional Commits:** Строгое следование стандарту (тип, scope, описание, тело, footer).
* **Превью перед сохранением:** Показывает итоговое сообщение перед выполнением команды `git commit`.
* **Без зависимостей:** Написано на чистом Python, использует только стандартные библиотеки. Не требует установки тяжелых пакетов.

---

## Установка

Утилита состоит всего из одного файла и легко интегрируется в вашу систему.

1. Скачайте скрипт в удобную для вас директорию (например, `~/scripts/`):
   ```bash
   mkdir -p ~/scripts
   curl -o ~/scripts/git_commit_helper.py [https://raw.githubusercontent.com/ВАШ_НИК/ВАШ_РЕПОЗИТОРИЙ/main/git_commit_helper.py](https://raw.githubusercontent.com/ВАШ_НИК/ВАШ_РЕПОЗИТОРИЙ/main/git_commit_helper.py)
   ```

2. Сделайте фаил испольным (для Linux / macOS):
   ```баш
   chmod +x ~/scripts/git_commit_helper.py
   ```

3. Создайте алиас (псевдоним) для быстрого вызова. Добавьте следующую строку в ваш `~/.bashrc` или `~/.zshrc`:
   ```баш
   псевдоним gch="python3 ~/scripts/git_commit_helper.py"
   ```

4. Обновите конфигурацию терминала:
   ```баш
   источник ~/.bashrc # источник или ~/.zshrc
   ```

## Как внести свой вклад (Внесение вклада)

Буду рад вашим Запросы на включение изменений! Если вы нашли баг или у вас есть идея для улучшения:

1. Форкните репозиторий
2. Создайте ветку для вашей фичи (`git checkout -b feature/AmazingFeature`)
3. Закоммитьте изменения (используя этот же скрипт)
4. Запушьте ветку (`Функция git push origin/AmazingFeature`)
5. Откройте запрос на извлечение
